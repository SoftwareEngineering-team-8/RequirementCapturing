# Use Case Descriptions

---

## Use Case Description: UC-01 (회원가입)

| Actor Action                                  | System Response                                        |
|-----------------------------------------------|--------------------------------------------------------|
|                                               | 1. 회원가입을 위한 필수 입력정보 폼과 회원가입 버튼을 보여준다.|
| 2. 본인의 ID, 비밀번호, 이름, 전화번호, 이메일을 입력한 후 회원가입 한다.|                                     |
|                                               | 3. 회원가입 완료 창을 출력한다.                           |

---

## Use Case Description: UC-02 (회원탈퇴)

| Actor Action             | System Response                                 |
|--------------------------|-------------------------------------------------|
|                          | 1. 회원탈퇴에 관한 안내사항과 회원탈퇴 버튼을 보여준다. |
| 2. 회원탈퇴 버튼을 클릭한다. |                                                 |
|                          | 3. 회원탈퇴 완료 창을 출력한다.

---

## Use Case Description: UC-03 (로그인)

| Actor Action                         | System Response                            |
|--------------------------------------|--------------------------------------------|
|                                      | 1. 로그인을 위한 아이디, 비밀번호 폼을 보여준다. |
| 2. 아이디, 비밀번호를 입력해 로그인 한다. |                                            |
|                                      | 3. 로그인 성공/실패 메세지를 출력한다.           |

---

## Use Case Description: UC-04 (설문 등록)

| Actor Action                         | System Response                            |
|--------------------------------------|--------------------------------------------|
|  | 1. 설문 등록에 필요한 정보를 입력하는 칸을 보여준다. |
| 2. 설문 제목, 설명, 문항, 응답 항목, 시작 시간, 마감 시간을 입력한다.| |
| | 3. 등록 확인 창을 출력한다. |   

---

## Use Case Description: UC-05 (설문 조회)

| Actor Action                         | System Response                            |
|--------------------------------------|--------------------------------------------|
|                                      | 1. 등록된 모든 설문을 보여준다. 각 설문 옆에 상세 보기와 삭제 버튼을 같이 출력한다.|
|                                      |                                            |

### Extensions
| Actor Action                         | System Response                            |
|--------------------------------------|--------------------------------------------|
| 2a. 상세 버튼을 선택한다.               |                                            |
|                                      | 3a. 선택한 설문에 해당하는 상세 정보를 팝업창으로 출력한다. | 
| 2a. 삭제 버튼을 선택한다.               |                                            | 
|                                      | 3b. 삭제 완료 확인창을 출력한다.               | 
---

## Use Case Description: UC-06 (설문 검색)

| Actor Action                         | System Response                            |
|--------------------------------------|--------------------------------------------|
|                                      | 1. 설문 검색에 필요한 조건을 보여준다.          |
| 2. 진행중, 진행 예정, 종료 중 하나를 선택하고 키워드를 입력한다.|                         |
|                                      | 3. 조건에 맞는 설문 리스트를 출력한다.          |   

---

## Use Case Description: UC-07 (설문 상세정보 조회)

| Actor Action                         | System Response                            |
|--------------------------------------|--------------------------------------------|
| | 1. 선택한 설문의 설문 제목, 설명, 문항, 응답 항목, 시작 시각 및 마감 시각 등이 출력된 상세 정보 화면을 보여준다. 현재 진행 중인 설문의 경우 응답 버튼을 같이 출력한다. | 

---

## Use Case Description: UC-08 (설문 응답)

| Actor Action                         | System Response                            |
|--------------------------------------|--------------------------------------------|
| | 1. 선택한 설문의 응답 화면을 보여준다. |
| 2. 각 문항에 대한 응답을 입력한 후 응답 완료 버튼을 클릭한다.| |
| | 3. 입력한 응답 내용을 저장하고 완료 메시지를 출력한다. |  

---

## Use Case Description: UC-09 (설문 응답 통계 정보 조회)

| Actor Action                         | System Response                            |
|--------------------------------------|--------------------------------------------|
| | 1. 통계 조회를 위한 검색 단위 선택 항목(최근 1주일, 1개월, 1년)을 보여준다. |
| 2. 검색 단위를 선택한다. | |
| | 3. 선택한 단위에 대한 전체 설문 수, 종료된 설문 수, 총 응답 수를 출력한다. |
