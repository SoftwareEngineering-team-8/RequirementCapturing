# Requirement List (functional requirement만 작성함)

| No. |                     Requirement                                | Use Case(s) |
|-----|----------------------------------------------------------------|-------------|
|  1  | 회원의 필수 입력 정보는 ID,비밀번호, 이름, 전화번호, 이메일이다          | 회원가입 |
|  2  | 회원은 언제든 탈퇴할 수 있으며, 탈퇴 시 모든 이용 권한과 데이터가 삭제된다 | 회원탈퇴 |
|  3  | 관리자와 회원은 ID와 비밀번호로 로그인한다                             | 로그인   |
|  4  | 새로운 설문을 등록하기. 여기에는 제목, 설명, 문항, 응답 항목, 시작 시간, 마감 시간 등을 포함한다. | 설문 등록 |
|  5  | 등록된 모든 설문 조회하기. 조회 시 특정 설문의 상세 정보를 확인하거나 삭제 할 수 있다. | 설문 조회 |
|  6  | 조건을 만족하는 설문 검색하기. 조건에는 진행중, 진행 예정, 종료, 키워드 등이 있다.  | 설문 검색 |
